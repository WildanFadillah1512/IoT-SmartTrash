## Overview

SmartTrash adalah sistem IoT pintar untuk memonitor kapasitas tempat
sampah secara otomatis. Menggunakan sensor ultrasonik, mikrokontroler
ESP, dan koneksi cloud, sistem ini mampu mengirimkan status tempat
sampah secara real-time ke dashboard maupun WhatsApp alert.

## Features

-   Real-time Monitoring\
-   Kirim data via WiFi ke Cloud\
-   Firebase Realtime / Thinger.io\
-   Notifikasi WhatsApp otomatis\
-   Mode otomatis & manual\
-   Scalable untuk Smart City
